# Project Brief: HireSphere v5 (ResumeMatchV2)

## The 2026 Talent Paradigm
The 2026 recruitment landscape has transitioned from reactive coding to "Product Engineering," where elite talent is evaluated on their ability to architect autonomous digital workforces. This project addresses the global hiring "white space" by treating agents as autonomous actors capable of multi-step planning and self-validation[cite: 1].

## Core Objective
Build a **Universal Agentic Engine** leveraging **Durable Execution** and **Factual Grounding** to automate the end-to-end recruitment lifecycle. The system must support asynchronous workflows where agents handle 30-minute tasks independently.

## MoSCoW Prioritization
*   **Must Have**: 
    *   Asynchronous multi-agent orchestration via Google Antigravity Mission Control[cite: 1, 2].
    *   Atomic factual grounding using **SROM (Self-Reflexive Ontology Mapping)** atoms[cite: 1, 2].
    *   $O(d \cdot \log N)$ retrieval complexity for multi-million vector datasets.
*   **Should Have**: 
    *   Thermal-aware **"Green Code"** for Samsung S24 Ultra (NPU-offloading) and FastAPI backends[cite: 1, 2].
    *   **Checkpointed Recovery** logic using **P-HNSW** for crash-consistent states.
*   **Could Have**: "Vibe Designing" via Google Stitch; Integrated Chrome instances for autonomous UI testing[cite: 1].

## Success Metrics
*   **SWE-bench**: Achieve >75% resolution rate on verified tasks[cite: 2].
*   **Governance**: 100% audit trail compliance for financial decisions using SROM fact-level verification[cite: 2].
*   **Efficiency**: Sub-50ms retrieval latency on 1M+ vector embeddings[cite: 2].
